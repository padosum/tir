## 🤔 생각하기  
- 요즘 개발 공부를 하고 테니스를 배우며 도무지 느는 것 같지가 않아 내 자신이 참 답답한 상황이다.  
  - 근데 이런 와중에 이런 글을 발견하게 되어 매우 기쁘다.  
- 프로그래밍이 쉬운 것이라 생각하고 그걸 바탕으로 쉽게 내가 해내지 못한 이유를 찾고 바꿔나가야겠다.
