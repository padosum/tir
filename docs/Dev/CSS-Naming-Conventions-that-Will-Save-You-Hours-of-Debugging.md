## 🤔 생각하기     
- 다른 사람의 코드를 보다가 class에 `__`를 붙이는 걸 보았다. `BEM`이라는 설명을 했고 그게 뭔지 검색하다가 발견한 글이다.  
- 디버깅을 위해 class 이름을 그냥 생각나는데로 짓는 것이 아니라 무엇을 하는지, 또 어디서 사용하고 class 들의 관계를 알 수 있게 하면 개발을 더 수월하게 할 수 있게 돕는 것이라는 걸 알게되었다.  
  - 내가 나중에 직장에 들어가서 소속된 팀에 따라 컨벤션이 다르겠지만 이러한 규칙을 사용하면 어떠한 이점이 있는지, 또 주의사항도 미리 알 수 있어서 좋았다.
