## 📝 요약 
- 플랫폼별 웹 개발자 도구의 역사  

## 🤔 생각하기 
- JavaScript와 웹은 뗄래야 뗄 수 없기 때문에 항상 관련 서적을 보면 웹 개발 역사 같은 것들이 나온다.  
  - 거기 나온 내용들과 비슷한 내용의, 또 알아듣기 쉬운 이야기로 풀어준 글이다.  
- 많은 역사 속 이야기 처럼, 경쟁 구도가 있었기에 이렇게 편리한 개발 도구들이 만들어진 것 같다.  
  - 글 끝부분 처럼, 소중하고 감사하게 생각하고 열심히 활용해야겠다.
