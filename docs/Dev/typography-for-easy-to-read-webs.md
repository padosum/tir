# 🖥 읽기 쉬운 웹을 위한 타이포그래피

- 링크: [https://parksb.github.io/article/37.html](https://parksb.github.io/article/37.html)

## 📝 요약 
- 조판 원칙을 통해 가독성을 높여 읽기 쉬운 웹을 만드는 법 

## 🤔 생각하기 
- 웹에서 폰트를 단순히 불러오고 `font-family`를 지정하는 방법 밖에 몰랐는데 가독성을 위해선 많은 작업이 필요하다는 것을 알게되었다.
