## 📝 요약

- 코드 리뷰에서 수행해야 할 것들을 "피라미드" 모양으로 정리한 그림

## 🤔 생각하기

- 친구와 코드 리뷰를 진행하려고 해서 알아보던 중에 발견했다. 잘 참고해서 진행하도록 해야겠다.
- 코드 리뷰를 할 때 그림에 나온 것처럼 코드스타일, 테스트는 자동화하고 구현, 문서화, API를 리뷰하는데 집중하자!
